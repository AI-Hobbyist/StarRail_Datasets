### 注释

| 文件夹名称 |                      用途                       |
| :--------: | :---------------------------------------------: |
|   Index    |        星穹铁道语音的索引文件，用于分类         |
|   Label    | 标注文件，目前提供 DifffSinger 标注和 Vits 标注 |
|   Voice    |                 数据集下载地址                  |
